# Uno-Desenvolvimento-Web
Trabalho referente a matéria Desenvolvimento Web, do 5 semestre, alunos Edgar Jr. Berlanda e Pedro Lucas Marció.

O trabalho se refere a um sistema baseado em marketplace, voltado a cultura familiar rural e pequenos e médios produtores, os conectando diretamente com seu consumidor final. O projeto vai consistir em um sistema de postagem de anúncios, caixa de mensagens e em certos casos sistema de pagamentos, para checkout de compras dentro do aplicativo.
